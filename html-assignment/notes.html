List of HTML tags

h1
It indicates the first heading.The most important heading on page.
p
It uses when we want to write a paragraph or text.
ol
It indicates an ordered list. Inside this tag we use li tag.
cite
It uses for citing sources. To cite we should add a blockquote tag first.
br
It creates a line break. It is important where division of lines is needed.
